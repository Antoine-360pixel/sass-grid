sass-grid
=========

## Current Version : v1.0

## Usage
- no compass require
- just init variables

##Example 
```html
<main class="fluid-row">
  <section class="col-3"></section>
  <section class="col-3"></section>
</main>
```

```scss
@import "sq_grid";

.box {
  @extend col(value);
}
```
